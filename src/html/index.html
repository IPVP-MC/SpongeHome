    About

    IPVP is a hardcore Minecraft server that was created in early 2015. Originating as HCFPVP, it features Hardcore Factions as its primary game.

    The primary concept behind IPVP is to create a rich and competitive environment for players to battle eachother. The server features many custom plugins, events, and features to enhance the hardcore experience.

    With new features constantly being worked on, we strive to be on the cutting edge of hardcore servers with unique gameplay aspects.
